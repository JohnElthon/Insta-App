# Insta-App

Aplicativo mobile desenvolvido em React Native inspirado na interface do Instagram.

## Sobre o projeto

O **Insta-App** foi desenvolvido como prática de interface e listagem de conteúdo em React Native.

O objetivo foi criar uma tela inspirada no feed do Instagram, trabalhando com componentes, imagens, listas, dados e estilização de elementos visuais.

Entre os dois repositórios de Instagram, este foi escolhido como a versão mais interessante para manter pública, por parecer uma versão mais personalizada do projeto.

## Funcionalidades

- Interface inspirada no Instagram
- Exibição de feed
- Listagem de postagens
- Uso de imagens locais e/ou externas
- Organização de componentes
- Prática com listas em React Native
- Estilização de tela mobile

## Tecnologias utilizadas

- React Native
- JavaScript
- Expo
- FlatList
- Node.js / npm

## Estrutura do projeto

O projeto possui uma estrutura organizada com os arquivos principais da aplicação, incluindo:

- `assets` - arquivos e recursos visuais
- `src` - componentes, imagens e listas
- `App.js` - arquivo principal do aplicativo
- `package.json` - dependências e scripts do projeto

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/JohnElthon/Insta-App.git
```

2. Acesse a pasta do projeto:

```bash
cd Insta-App/Insta-App
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```

5. Abra no emulador Android/iOS ou no aplicativo Expo Go.

## Observação

Projeto desenvolvido com fins acadêmicos para praticar criação de interfaces, componentização e renderização de listas em React Native.
